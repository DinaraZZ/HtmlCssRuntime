    BASE

1) блочная группа - занимает всю ширину
2) линейная группа - только ширину своего наполнения

<p></p> - блочный текст контейнер
<span></span> - линейный

<a></a> - ссылка на перенаправление. Необходимо задавать аттрибут href (ссылка), target (в какой вкладке): _self,
_blank

table: thead, tbody, tfoot
tr (ряд), th, td (ячейки)
colspan="кол ячеек" - объедин по горизонтали
rowspan - по вертикали

<img src="" alt=""/> - добавление картинки, alt - текст вместо картинки, если не прогрузится


    FORMS

Позволяет пользователю ввести данные.
action - адрес, на который будут отправлены данные,
method - каким способом.

Аттрибут name обязателен для отправки информации.
disabled блокирует ввод значения, без отправки
readonly - запрещает изменять, с отправкой
value - значение в этих случаях по умолчанию

<input type="text">
<button type="submit"></button> отправка формы

<input type="password">

<textarea></> - несколько строчек

<input type="radio"> - выбор одного варианта
для группировки должен быть указан одинаковый name
value - название для корректной отправки
checked - сделает вариант выбранным

При взаимодействии с текстом (label), будет задействован input

<input type="checkbox"> - выбор нескольких вариантов

<button type="reset"></button> сбросить формы
