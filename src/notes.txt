    CSS

Типы селекторов:
1) по названию хтмл элемента - <a href="/"></a> -> a
    a {} - все элементы a
    div a - родители div
    div > a - прямой родитель
2) по классификатору (аттрибут class) - перед названием ставится точка в css
    .link - все элементы link
    .menu .link - родители menu
    .menu > .link - прямой родитель
    a.link - только у a
    .link.primary - применены оба класса
Если у одного хтмл элемента несколько классов с одним свойством - берется последний
3) по идентификатору (аттрибут id)

margin - отступ снаружи
margin-left: auto - прилипнет вправо
+right: auto / margin: 0 auto - центр

padding - внутренний отступ. суммируется с размером элемента.
content-box: суммируется
border-box: не суммируется

* {
    margin: 0;
    padding: 0;
    outline: none;
} - убирает все значения по умолчанию

border - внутренняя рамка
outline - внешняя рамка

box-shadow: 0 0 0px rgba(0,0,0,0.7)
box-shadow: inset ...

font-size
font-weight: [100-900]
color:
text-align:
text-decoration:

        ПСЕВДО-КЛАССЫ
Дополнительные селекторные применяемые к основному.
:hover - наведение на элемент курсором
:active - нажатие на элемент ЛКМ (пока в нажатом состоянии)
:focus - фокусировка на элементе (интерактивные элементы: input, textarea, select)
:focus-within - у внутренних классов сработал focus, класс задается для родителя

::first-letter - работает только в блочных (флекс) элментах (родитель)
